Any FieldSig has three main components: (EMCA-335 page 288)
- FIELD (which is the literial int 0x6)
- CustomMod (which can be one or more)
- Type 

CustomMod and Type are also signatures

For more info on CustomMod and Type please refer to the documentation
the corresponding documentation is at
CustomMod (ECMA-335 page 291) 
Type (ECMA-335 page 289)
